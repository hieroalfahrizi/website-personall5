# USK-Hiero Website

## Deskripsi
USK-Hiero adalah situs web statis yang menyediakan informasi tentang profil, layanan, dan kontak untuk organisasi atau institusi terkait. Situs ini dibangun menggunakan HTML, CSS, dan JavaScript untuk memberikan pengalaman pengguna yang sederhana dan responsif.

## Struktur File
- `index.html` - Halaman utama situs web
- `profil.html` - Halaman profil organisasi
- `layanan.html` - Halaman layanan yang ditawarkan
- `kontak.html` - Halaman informasi kontak
- `assets/css/style.css` - File stylesheet untuk styling halaman
- `assets/js/script.js` - File JavaScript untuk interaktivitas
- `assets/images/` - Folder berisi gambar-gambar yang digunakan di situs

## Cara Menjalankan
1. Pastikan Anda memiliki browser web (seperti Chrome, Firefox, atau Edge).
2. Buka file `index.html` di browser Anda dengan mengklik dua kali pada file tersebut atau menariknya ke jendela browser.
3. Navigasi melalui situs menggunakan menu atau tautan yang tersedia.

## Teknologi yang Digunakan
- **HTML**: Untuk struktur dan konten halaman
- **CSS**: Untuk styling dan tata letak
- **JavaScript**: Untuk fitur interaktif (jika ada)
